# Cancer Analytics Project

End-to-end data analytics project analyzing cancer cases and industry impact.

## Tools Used
- Python (Jupyter Notebook)
- SQL Server (SSMS)
- Power BI
- Git & GitHub

## Project Workflow
1. Data generation and cleaning using Python
2. Data analysis using SQL
3. Dashboard creation using Power BI

## Key Analysis
- Cancer cases by district
- Distance-based exposure analysis
- Industry and pollutant impact
